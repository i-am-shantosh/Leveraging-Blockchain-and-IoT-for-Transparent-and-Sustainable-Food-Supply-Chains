Leveraging Blockchain and IoT

For Transparent and Sustainable Food Supply Chains

Project Overview

This research explores how blockchain and IoT technologies can be integrated to ensure transparency, sustainability, and trust in food supply chains. By combining immutable ledger systems with real-time sensor data, the solution aims to prevent fraud, reduce waste, and ensure traceability from farm to table.

Key Technologies

  • Hyperledger Fabric (Private Blockchain)
  • IoT Devices for Environmental Monitoring
  • Smart Contracts for Secure Transactions
  • Web Interface for Real-Time Data Visualization
